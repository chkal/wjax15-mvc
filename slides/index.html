JAX-RS

JAX-RS Fakten

  • "Java API for RESTful Web Services"
  • Erstes Release 2008
  • Seit 1.1 Bestandteil von Java EE 6
  • JSR 2.1 -> JSR 370 -> Java EE 8

JAX-RS Beispiel #1


@Path("/hello")
public class SimpleResource {

  @GET
  public String hellWorld() {
    return "Hello world";
  }

}

JAX-RS Beispiel #2


@Path("/hello")
public class SimpleResource {

  @GET
  @Path("/{name}")
  public String sayHello( @PathParam("name") String name ) {
    return "Hello " + name;
  }

}
